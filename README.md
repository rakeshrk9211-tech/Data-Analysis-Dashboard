# 📊 Vrinda Store – Annual Report 2022 Dashboard(Interactive Dashboard creation using MS Excel)
## Project Objective 
This is an **end-to-end data analytics project** built in Microsoft Excel for **Vrinda Store**. The dashboard analyzes 2022 sales data to provide actionable business insights.
## Dataset used 
- <a href="https://github.com/rakeshrk9211-tech/Data-Analysis-Dashboard/blob/main/Vrinda%20Store%20Data%20Analysis.xlsx">Dataset</a>

**Objective:** Transform raw store data into an interactive dashboard that helps stakeholders understand:
- Sales trends by month & state
- Gender-based purchasing patterns
- Channel performance (Amazon vs Flipkart)
- Order status breakdown

  - Dashboard Interaction <a href="https://github.com/rakeshrk9211-tech/Data-Analysis-Dashboard/blob/main/Vrinda_store.jpg">View Dashboard</a>

  ## Process
  - Verify data for any missing values and anomalies, and sort out the same.
  - Made sure data is consistent and clean with respect to data type, data format and values used.
  - Created pivot tables according to the questions asked.
  - Merge all pivot tables into one dashboard and apply slicer to make dynamic.
